# Ochi Website Clone

A modern and interactive clone of the Ochi website homepage built to showcase front-end development skills. This project focuses on smooth animations, engaging user interactions, and clean UI design using GSAP.

## 🚀 Features

- Smooth page animations with GSAP
- Interactive text and hover effects
- Modern responsive landing page design
- Clean and reusable code structure
- Scroll-triggered animations
- Custom typography and layout inspired by Ochi

## 🛠️ Technologies Used

- React
- CSS3
- JavaScript (ES6)
- GSAP (GreenSock Animation Platform)

## 🎯 Purpose

The goal of this project was to recreate the visual experience of the Ochi homepage while practicing advanced front-end animation techniques and improving UI/UX development skills.

## 📚 What I Learned

- Creating smooth animations using GSAP
- Working with timelines and animation sequencing
- Building responsive layouts
- Improving user engagement through motion design
- Structuring scalable front-end projects
